Bienvenido al sitio web de Winlator
Winlator es una aplicación para Android que te permite ejecutar aplicaciones de Windows (x86_64) con Wine y Box86/Box64.

Palabras De Bruno

Estoy inmensamente agradecido a ptitSeb y a los desarrolladores de Wine, quienes desarrollaron la capa de compatibilidad de Windows (x86_64). Sin la dedicación constante a Box86/Box64 por parte de ptitSeb y las constantes mejoras a Wine por parte de sus desarrolladores, este proyecto nunca habría sido posible.
